# Clawbot staging

Estructura base para futura integración de Clawbot/OpenClaw en el entorno staging de Verity News.

## Estado
- aún no desplegado
- aún sin imagen/contenedor
- aún sin acceso a red adicional
- aún sin webhook configurado

## Principios
- mínimo privilegio
- sin docker.sock
- sin privileged
- sin host network
- sin puertos públicos
- salida controlada por webhook en fase posterior
