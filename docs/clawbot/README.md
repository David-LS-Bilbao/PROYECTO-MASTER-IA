# Clawbot Secure Ops

Documentación base de la feature `feat/clawbot-secure-ops`.

## Objetivo
Integrar Clawbot/OpenClaw como capa confinada de supervisión e investigación para Verity News, sin alterar producción ni conceder privilegios peligrosos.

## Principios
- mínimo privilegio
- aislamiento por defecto
- sin escritura productiva directa
- sin acceso a secretos de producción
- reversibilidad completa
- revisión humana obligatoria
