# Operating Rules

## Permitido
- descubrir fuentes candidatas
- revisar dominios y feeds públicos
- resumir estado operativo
- enviar informes estructurados a un webhook controlado
- ejecutar chequeos programados simples

## Prohibido
- actuar como crawler principal
- escribir en producción
- usar exec libre
- acceder a paneles internos
- conectarse a PostgreSQL productivo
- sustituir a n8n o al backend
